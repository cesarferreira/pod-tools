## Installation

Install it via terminal:

    $ gem install pod-tools

## Usage

On your iOS project folder:

Clean cocoapods data
```bash
pod-clean
```

Clean cocoapods data and Reinstall
```bash
pod-reinstall
```

Clean all XCode derived data
```bash
pod-clean-derived-data
```
